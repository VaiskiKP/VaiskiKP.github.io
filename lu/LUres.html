icons

LU handels icons in a way that they can be upcaled as much as you want which is the reason the ui doesn't same the same size when you make the resolution higher or lower. This means that whatever texture you change the default icon into the game will resize it to a its original shape and size. This means if the originl icon was 32x32px and you changed it to 100x2000px it would stretch it into 32x32px.
